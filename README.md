# Stashd Investor Demo Website

This is a static demo website for investor pitches. It has no build step and can be hosted directly on GitHub Pages.

## Files

- `index.html` — main site
- `styles.css` — visual system and responsive layout
- `app.js` — tab switching and phone screen rendering
- `favicon.svg` — simple Stashd favicon
- `.nojekyll` — tells GitHub Pages not to run Jekyll

## Publish on GitHub Pages

1. Create a new GitHub repository, for example `stashd-investor-demo`.
2. Upload all files in this folder to the root of the repository.
3. Commit the files to the `main` branch.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Save.
7. GitHub will generate a public URL, usually:
   `https://YOUR-USERNAME.github.io/stashd-investor-demo/`

## Local preview

Open `index.html` directly in your browser.
