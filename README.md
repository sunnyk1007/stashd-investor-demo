# Stashd Investor Demo Website

Static GitHub Pages site for investor meetings.

## Files

- `index.html` — main site with direct-rendered demo screens
- `styles.css` — visual system and responsive layout
- `app.js` — tab switching only
- `favicon.svg`
- `.nojekyll`

## Publish / Update on GitHub Pages

1. Upload or replace all files in the root of your GitHub repo.
2. Commit changes to `main`.
3. GitHub Pages will automatically redeploy.
4. Your public URL should remain:
   `https://sunnyk1007.github.io/stashd-investor-demo/`

## Note

This version embeds all phone demo screens directly in `index.html`, so the Teen App and Parent App screens should show even if JavaScript rendering is limited. JavaScript is only used for tab switching.
